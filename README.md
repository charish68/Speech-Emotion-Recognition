# Speech Emotion Recognition (Starter)


Simple Speech Emotion Recognition (SER) project: upload a short speech audio file (wav/mp3) and get the predicted emotion with probabilities via a Flask API. This starter includes code for MFCC feature extraction, a small CNN, training script, and a minimal frontend demo.


## Quickstart (local)


1. Clone or create repo and copy files from this starter.


2. Create virtual environment and install:


```bash
python -m venv venv
# mac/linux
source venv/bin/activate
# windows
